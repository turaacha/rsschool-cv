Employee & Project Dashboard 📋

Удобная панель управления для учета сотрудников, планирования проектов и расчета прибыли. Приложение позволяет управлять ресурсами компании помесячно, сохраняя историю данных.

Основные функции
* **Dashboard:** Переключение между списками проектов и сотрудников.
* **Финансы:** Автоматический расчет выручки, затрат и чистой прибыли.
* **Календарь:** Учет отпусков и их влияния на доходность проекта.
* **Seed Data:** Быстрое копирование данных из прошлых месяцев для планирования.
* **Управление:** Добавление, удаление и распределение сотрудников по проектам.

Технологии
* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (Vanilla JS)
* LocalStorage (сохранение данных в браузере)

Как запустить
1. Скачайте файлы проекта.
2. Откройте `index.html` в браузере.
3. Данные сохраняются автоматически — при обновлении страницы ничего не пропадет.

Структура проекта
* `index.html` — основная разметка.
* `style.css` — стили и анимации.
* `script.js` — логика расчетов и работа с данными.
